我需要开发一个系统，通过网页来来使用命令行工具，可以理解成一个网页的tmux，界面整体风格现在的 codex gui差不多，但是功能上有比较大的区别。

- 使用 portable-pty 结合 elixir 的 otp 来做 tty，在web上提供全能的命令行终端
- 使用 ash_typescript 来实现终端信息推送
- 终端产生的内容使用 DETS 做比较大的缓存（网页可以设置大小），这样可以刷新或者我断网重连可以重新看得到
- 需要一个抽屉式的文件管理器，可以看到当前pwd内的文件，方便快速查看
- 系统的帐号认证是可选择打开的，如果打开必须系统有设置好的帐号
- ash_authentication_phoenix 的界面要定制，符合统一的设计风格
